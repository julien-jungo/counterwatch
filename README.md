# Counterwatch

An Overwatch 2 counter picker. Browse heroes, search by name, and find out which heroes counter them.

**Live site:** [julien-jungo.github.io/counterwatch](https://julien-jungo.github.io/counterwatch)

## Tech Stack

- React 18
- React Router 6
- Tailwind CSS 3
- Create React App

## Getting Started

```bash
npm install
npm start
```

The app runs at [localhost:3000](http://localhost:3000).

## Scripts

| Command          | Description                      |
|------------------|----------------------------------|
| `npm start`      | Start the dev server             |
| `npm run build`  | Create a production build        |
| `npm run deploy` | Build and deploy to GitHub Pages |
